This project is a simple ASP.NET Web Forms application for managing a telephone diary. It allows users to store and manage contact information such as names, phone numbers, and addresses using a web-based interface connected to a SQL database. The system demonstrates basic web application development concepts including form handling, event-driven programming, and database integration.

Features

Add new diary records
Update existing records
View and select stored entries
Delete records by User ID
Display data using GridView
Navigation between multiple forms (Add, Update, Select, Delete)
Technologies Used

ASP.NET Web Forms
C#
SQL Server (Database)
HTML and Web Controls (Label, TextBox, Button, GridView)

Project Structure

TDiary1.aspx – Add new records
DiaryUpdate.aspx – Update existing records
SelectDiary.aspx – View/select records
DeleteDiary.aspx – Delete records
.sln – Visual Studio solution file
How to Run

Open the .sln file in Visual Studio
Make sure the database connection string is configured correctly
Run the project using IIS Express or a local server
Navigate through the pages using the provided links
Purpose
This project was developed as part of academic coursework to demonstrate understanding of CRUD operations, ASP.NET Web Forms, database connectivity, and basic user interface design.
